# Project Brief: Crazy Car

## Project Overview
Crazy Car is a web-based HTML5 driving game where players navigate a car through traffic, avoiding obstacles and collecting points.

## Core Requirements
- Single HTML file implementation (HTML + CSS + JavaScript)
- Canvas-based rendering for smooth 60fps gameplay
- Lane-based driving mechanics (3-4 lanes)
- Keyboard controls (arrow keys or A/D)
- Scoring system with high score persistence
- Increasing difficulty over time
- Responsive design for desktop browsers
- No external dependencies

## Goals
- Create an engaging, fast-paced arcade-style driving game
- Provide smooth gameplay with visual polish
- Implement intuitive controls and clear game states
- Achieve cross-browser compatibility
