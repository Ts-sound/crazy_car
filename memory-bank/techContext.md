# Tech Context: Crazy Car

## Technologies Used
- **HTML5** - Structure and canvas element
- **CSS3** - Styling and layout
- **JavaScript (ES6+)** - Game logic and rendering
- **HTML5 Canvas API** - 2D graphics rendering

## Development Setup
- Single file implementation (index.html)
- No build tools required
- No external dependencies
- Runs directly in browser
- Supports Chrome, Firefox, Safari, Edge

## Technical Constraints
- Must work without internet connection
- No external libraries (pure vanilla JS)
- Responsive but optimized for desktop
- Target 60fps performance
- Keyboard input only (no touch controls initially)

## Dependencies
- None (standalone HTML file)

## Tool Usage
- Standard text editor for development
- Browser for testing (open index.html directly)
- No server required
