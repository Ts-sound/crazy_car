# Product Context: Crazy Car

## Purpose
Crazy Car is a browser-based arcade driving game designed to provide quick, engaging entertainment. Players control a car navigating through traffic, testing their reflexes and reaction time.

## Problem Solved
Provides a simple, accessible game that requires no installation or external dependencies - just open in any modern web browser and play.

## User Experience Goals
- Instant playability with minimal learning curve
- Smooth, responsive controls
- Clear visual feedback
- Progressive difficulty to maintain engagement
- Satisfying gameplay loop with replay value

## How It Works
1. Player starts game from welcome screen
2. Uses left/right arrow keys or A/D to change lanes
3. Avoids incoming traffic/obstacles
4. Collects coins for bonus points
5. Game speeds up as score increases
6. Collision ends game with score display
7. Player can restart to try for higher score
